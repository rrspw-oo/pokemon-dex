# 寶可夢圖鑑搜尋器

一個基於 React 的寶可夢圖鑑搜尋工具。

## 功能特色

### 進化系統
- 自動顯示寶可夢的完整進化狀態
- 點擊寶可夢卡片重新搜尋該寶可夢的進化家族

### 強大搜尋功能
- 支援中文、英文名稱與編號搜尋
- 寶可夢編號快速查詢
- 模糊搜尋支援

## 使用說明

1. **搜尋寶可夢**: 在搜尋框輸入寶可夢名稱或編號
2. **查看進化鏈**: 搜尋結果自動包含該寶可夢的所有進化型態
3. **進化詳情**: 點擊任何寶可夢卡片查看以該寶可夢為主的進化鏈
4. **進化條件**: 每個進化階段都顯示詳細的進化需求

## 開發

```bash
# 安裝相依套件
npm install

# 開發模式
npm run dev

# 建置
npm run build
```

## API 參考

使用 [PokeAPI](https://pokeapi.co/) 獲取寶可夢資料和進化訊息。
